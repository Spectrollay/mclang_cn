<b>无需再忍受基岩版雷人翻译的困扰</b>

<b>最后更新: </b>2022/10/20<br>

<b>现已跟进:</b>
- Release 1.19.31
- Beta 1.19.50.21

- 中文译名修正
- 隐藏开发版水印
- 支持开发版独有内容
